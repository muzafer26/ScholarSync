# **App Name**: ScholarSync

## Core Features:

- Course Aggregation: Aggregated course listings from Coursera, Google, Microsoft, IBM, edX, Harvard, MIT, YouTube and Class Central.
- Opportunity Board: Display internship, fellowship, scholarship, and skill contest opportunities.
- Advanced Search: Enable search and filtering by category, provider, skill level, duration, and opportunity type.
- Location Awareness: Show opportunities based globally or specific to user location. The app should have an LLM-powered 'tool' to intelligently interpret location based on incomplete data.
- Bookmarking: Allow users to save resources for later review.

## Style Guidelines:

- Primary color: Dark indigo (#4B0082) to convey depth and sophistication, referencing academic tradition without being stuffy.
- Background color: Very light lavender (#F0F8FF) to provide a clean, unobtrusive backdrop that makes content pop.
- Accent color: Muted violet (#8A2BE2) for interactive elements, contrasting with the primary and background.
- Font pairing: 'Alegreya' (serif) for headings and 'PT Sans' (sans-serif) for body text. Alegraya will bring a humanist, intellectual and contemporary feel while PT Sans brings a modern, warm feel. This helps with readability.
- Use a consistent set of minimalist icons for categories and providers, aiding in quick visual identification.
- Employ a card-based layout to clearly present courses and opportunities, ensuring easy navigation and scanning.
- Use subtle transitions and loading animations to enhance user experience without being distracting.
