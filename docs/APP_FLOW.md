# ScholarSync Application Flow

## User Journey Map

1. **Home Page (`/`)**
   - User views the three primary immersive entry points.
   - User can log in or view their bookmarks from the header.

2. **Courses Page (`/courses`)**
   - Immersive 3D Book is displayed.
   - User clicks the right side of the book to flip forward.
   - User clicks the left side to flip back.
   - **Action:** Clicking "Start Course" on any page redirects the user to the provider's URL in a new tab.

3. **Opportunities Page (`/opportunities`)**
   - A closed 3D door is displayed.
   - User clicks the door to trigger an opening animation and glow effect.
   - A grid of opportunities appears after the animation completes.

4. **Guides Page (`/guides`)**
   - A shimmering treasure chest is displayed.
   - User clicks the chest. It shakes and then opens.
   - Career guides and roadmaps are revealed in a grid.

5. **Resource Interaction**
   - User clicks the Bookmark icon on any card to save it.
   - Toast notifications confirm the action.

6. **Bookmarks Page (`/bookmarks`)**
   - User views a grid of all resources they have saved.
   - If no bookmarks exist, a friendly empty state encourages them to explore.
